# "Swiftle" Example

https://elementary-swift.github.io/elementary-ui/

## Run Locally

Requires main or 6.2 snapshot with matching *Swift SDKs for WebAssembly* from [swift.org](https://www.swift.org/install).

```sh
npm install

# dev build + watch
npm run dev

# run  embedded release build
npm run build
npm run preview
```
